No Unity, abra o projeto na pasta `Unity`. O Unity pode solicitar uma versão diferente entre o projeto e a versão instalada em seu computador. Esse aviso não é incomum, contanto que sua versão do Editor do Unity seja mais recente do que aquele em que o projeto foi criado. Nesse caso, clique em **Continuar**. Caso sua versão do editor do Unity seja antiga do que a exigida pelo projeto, clique em **Sair**e atualize o editor do Unity.
