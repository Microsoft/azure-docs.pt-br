Uma VM (máquina virtual) recebe automaticamente um endereço IP privado de um intervalo que você especifica, com base na sub-rede na qual a VM é implantada. A VM retém o endereço até que a VM seja excluída. O Azure atribui dinamicamente o próximo endereço IP privado disponível da sub-rede na qual você cria uma VM. Se você quiser um endereço IP específico da sub-rede atribuída à VM, atribua um endereço IP estático.
