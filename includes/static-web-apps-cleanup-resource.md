Se você não quiser continuar usando esse aplicativo, poderá excluir o recurso Aplicativo Web Estático do Azure com as seguintes etapas:

1. Abra o [portal do Azure](https://portal.azure.com)
1. Na barra de pesquisa superior, pesquise seu aplicativo pelo nome fornecido anteriormente
1. Clique no aplicativo
1. Clique no botão **Excluir**
1. Clique em **Sim** para confirmar a ação
