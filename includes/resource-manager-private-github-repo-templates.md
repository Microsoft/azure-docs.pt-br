> Para implantar um modelo ou fazer referência a um modelo vinculado que é armazenado em um repositório GitHub privado, consulte uma solução personalizada documentada em um [blog do MVP](https://blog.tyang.org/2017/05/19/deploying-arm-templates-with-artifacts-located-in-a-private-github-repository/).
