1. Entre no servidor de processos como administrador.
2. Baixe a versão mais recente do [Instalador de Configuração Unificada](https://aka.ms/unifiedinstaller).
3. Clique duas vezes o instalador para iniciar o processo de atualização.
4. O instalador detecta os componentes do Site Recovery que estão instalados e os atualiza para a versão mais recente.
