Tenha as seguintes considerações de segurança em mente ao usar o Dispositivo de Nuvem StorSimple:

* O dispositivo de nuvem é protegido por meio de sua assinatura do Microsoft Azure. Isso significa que, se você estiver usando o dispositivo de nuvem e se sua assinatura do Azure estiver comprometida, os dados armazenados em seu dispositivo de nuvem também estarão suscetíveis.
* A chave pública do certificado usado para criptografar os dados armazenados no StorSimple foi disponibilizada com segurança para o portal do Azure e a chave privada é mantida com o Dispositivo de Nuvem StorSimple. No Dispositivo de Nuvem StorSimple, as chaves públicas e privadas são armazenadas no Azure.
* O dispositivo de nuvem é hospedado no datacenter do Microsoft Azure.
