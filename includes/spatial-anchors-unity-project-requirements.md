O SDK de âncoras espaciais do Azure deve ser configurado no <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019,4 (LTS)</a> usando os seguintes pacotes:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Realidade mista do Windows: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- Plug-in ARCore XR (para suporte a dispositivos Android): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- Plug-in ARKit XR (para suporte a dispositivos iOS): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> O SDK do ASA atualmente não dá suporte à <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">estrutura de plug-ins do Unity XR</a>
