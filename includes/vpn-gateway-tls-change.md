Começando em 1 de julho de 2018, o suporte está sendo removido para TLS 1.0 e 1.1 do Gateway de VPN do Azure. O Gateway de VPN oferecerá suporte somente ao protocolo TLS 1.2. Somente conexões de ponto a site são afetadas; conexões site a site não serão afetadas. Se você estiver usando o TLS para VPNs ponto a site em clientes Windows 10, você não precisa realizar nenhuma ação. Se você estiver usando o TLS para conexões ponto a site em clientes Windows 7 e Windows 8, consulte o [perguntas frequentes sobre o Gateway de VPN](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) para obter instruções de atualização.
