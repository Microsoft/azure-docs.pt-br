> O uso de uma chave de segurança como um método de autenticação sem senha está atualmente em versão prévia pública. Se o que você está vendo na tela não corresponde ao que está sendo abordado neste artigo, isso significa que o administrador não ativou este recurso ainda. Até que este recurso seja ativado, você precisará escolher outro método de autenticação na página [**Informações de Segurança**](../articles/active-directory/user-help/security-info-setup-signin.md). Para saber mais sobre versões prévias, consulte os [Termos de Uso Complementares para Visualizações do Microsoft Azure](https://azure.microsoft.com/support/legal/preview-supplemental-terms/).
