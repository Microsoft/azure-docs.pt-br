***Antes de começar***, use o seletor acima para escolher o tipo de política que está configurando. O Azure AD B2C oferece dois métodos para definir como os usuários interagem com seus aplicativos: por meio de [fluxos dos usuários](../articles/active-directory-b2c/user-flow-overview.md) predefinidos ou [políticas personalizadas](../articles/active-directory-b2c/custom-policy-overview.md) totalmente configuráveis. As etapas necessárias neste artigo são diferentes para cada método.
