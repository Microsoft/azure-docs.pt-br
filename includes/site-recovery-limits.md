Os seguintes limites se aplicam ao Azure Site Recovery.

| Identificador de limite | Limite |
| --- | --- |
| Número de cofres por assinatura |500 |     
| Número de servidores por cofre dos Serviços de Recuperação |250 |
| Número de grupos de proteção por cofre dos Serviços de Recuperação |Sem limite |
| Número de planos de recuperação por cofre dos Serviços de Recuperação |Sem limite |
| Número de servidores por grupo de proteção |Sem limite |
| Número de servidores por plano de recuperação |100 |
