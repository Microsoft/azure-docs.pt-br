> Atualmente, a <a a href="https://aka.ms/MRFeatureToolDocs" target="_blank">Ferramenta de Recurso de Realidade Misturada</a> é compatível apenas com o Windows.

1. Consulte a documentação da <a href="https://aka.ms/MRFeatureToolDocs" target="_blank">Ferramenta de Recurso de Realidade Misturada</a> para configurá-la e aprender a usá-la.
2. Instale o pacote de recursos "Âncoras de Objeto do Microsoft Azure" na pasta do projeto do Unity.
