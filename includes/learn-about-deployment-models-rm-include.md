> O Azure tem dois modelos de implantação diferentes que você pode usar para criar e trabalhar com recursos: [Azure Resource Manager e clássico](../articles/azure-resource-manager/management/deployment-models.md). Este artigo abordará o uso do modelo de implantação do Resource Manager. Recomendamos usar o modelo de implantação do Resource Manager para executar novas implantações em vez do modelo clássico de implantação.
