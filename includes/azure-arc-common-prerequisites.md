## <a name="prerequisites"></a>Pré-requisitos

Antes de prosseguir com as tarefas deste artigo, você precisa ter:

- [As ferramentas de cliente instaladas](../articles/azure-arc/data/install-client-tools.md)

   As ferramentas incluem `azdata` , Azure Data Studio e CLI do Azure.

   Para Azure Data Studio, instale as extensões para a **CLI de dados do Azure**, o **arco do Azure**e o **PostgreSQL** (dependendo dos serviços de dados)

- [Um controlador de dados de arco do Azure instalado](../articles/azure-arc/data/create-data-controller.md)
