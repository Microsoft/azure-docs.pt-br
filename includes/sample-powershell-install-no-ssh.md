Este exemplo exige o Azure PowerShell Az 1.0 ou posterior. Execute `Get-Module -ListAvailable Az` para ver quais versões estão instaladas. Se é preciso instalar, consulte [Instalar o módulo do Azure PowerShell](/powershell/azure/install-az-ps). 

Execute [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) para entrar no Azure.
