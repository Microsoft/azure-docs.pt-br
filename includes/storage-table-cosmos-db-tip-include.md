> [!TIP]
> O conteúdo deste artigo se aplica ao armazenamento de Tabelas do Azure original. No entanto, agora há uma oferta alternativa para o armazenamento de tabelas: a API de Tabela do Azure Cosmos DB. Essa API oferece maior desempenho e disponibilidade, distribuição global e índices secundários automáticos. Ela também está disponível em um modo [sem servidor](../articles/cosmos-db/serverless.md) baseado em consumo. Existem algumas [diferenças de recursos](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) entre a API de Tabela no Azure Cosmos DB e no armazenamento de tabelas do Azure. Para obter mais informações, consulte o [API de Tabela do Azure Cosmos DB](../articles/cosmos-db/table-introduction.md).
