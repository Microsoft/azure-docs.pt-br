- **[Monitorar com painéis de Azure Data Studio](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> O Azure Data Studio é uma ferramenta de banco de dados multiplataforma para profissionais de dados que usam a família Microsoft de plataformas de dados locais e na nuvem em Windows, MacOS e Linux. O Azure Data Studio oferece uma experiência de editor moderna com IntelliSense, snippets de código, integração de controle do código-fonte e um terminal integrado. Ele é projetado pensando no usuário da plataforma de dados, com plotagem de conjuntos de resultados de consulta e painéis personalizáveis internos.

**[Monitorar com o Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana é uma análise de software livre de várias plataformas e um aplicativo Web de visualização interativa. Ele fornece gráficos, grafos e alertas para a Web quando conectados a fontes de dados com suporte. Ele é expansível por meio de um sistema de plug-in. Os usuários finais podem criar painéis de monitoramento complexos usando a consulta interativa.

**[Log Analytics com Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana é um painel de visualização de dados de software livre para Elasticsearch. Ele fornece recursos de visualização sobre o conteúdo indexado em um cluster Elasticsearch. Os usuários podem criar gráficos de barras, de linhas e de dispersão ou de pizza e mapas sobre grandes volumes de dados.
