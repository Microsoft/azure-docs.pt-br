É possível criar e gerenciar VMs (máquinas virtuais) em um dispositivo Azure Stack Edge usando APIs. Essas são APIs padrão do Azure Resource Manager, chamadas usando o ponto de extremidade do Azure Stack Edge local. As APIs de Azure Resource Manager fornecem uma camada de gerenciamento consistente que, nesse caso, permite que você crie, atualize e exclua VMs em uma assinatura local que existe no dispositivo. Com o uso de cmdlets do Azure PowerShell, é possível conectar-se ao Azure Resource Manager em execução no Azure Stack Edge.
