## <a name="version-the-device-template"></a>Realizar o controle de versão do modelo de dispositivo

O aplicativo de Análise de vídeo – detecção de objetos e de movimento inclui o modelo de dispositivo **Gateway de LVA do Edge**. Para usar o arquivo de manifesto de implantação atualizado com esse modelo de dispositivo, você precisa criar e publicar uma versão do modelo.

Para criar uma versão do modelo:

1. No aplicativo do IoT Central, navegue para **Modelos de Dispositivo** e selecione o modelo de dispositivo **Gateway de LVA do Edge**.

1. Selecione **Versão** para criar um modelo chamado **Gateway de LVA do Edge v2** e, em seguida, selecione **Criar**.

O novo modelo de dispositivo **Gateway de LVA do Edge v2** agora está em sua lista de modelos de dispositivo.
