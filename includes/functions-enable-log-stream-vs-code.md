Para ativar os logs de streaming para seu aplicativo de funções no Azure:

1. chave de pressione F1 para abrir a paleta de comandos, em seguida, procure e execute o comando **Azure Functions: Iniciar o Streaming de Logs**.

1. Selecione seu aplicativo de funções no Azure e selecione **Sim** para habilitar o log de aplicativo para seu aplicativo de funções.

1. Disparar as funções no Azure e observe que os dados de log são exibidos na janela de saída no Visual Studio Code.

1. Quando terminar, lembre-se de executar o comando **Azure Functions: Parar os Logs de Streaming** para desabilitar o log para o aplicativo de funções.
