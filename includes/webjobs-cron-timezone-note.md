> [!NOTE]  
> O fuso horário padrão usado para executar expressões CRON é o UTC (Tempo Universal Coordenado). Para que a expressão CRON seja executada com base em outro fuso horário, crie uma configuração de aplicativo para o aplicativo de funções denominada WEBSITE_TIME_ZONE. Para saber mais, confira [Fusos horários do NCRONTAB](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
