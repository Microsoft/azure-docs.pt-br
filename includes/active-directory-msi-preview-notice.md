Identidades gerenciadas para recursos do Azure é um recurso do Azure Active Directory. Cada um dos [serviços do Azure que dão suporte a identidades gerenciadas para recursos do Azure](../articles/active-directory/managed-identities-azure-resources/services-support-managed-identities.md) está sujeito à própria linha do tempo. Não deixe de examinar o status de [disponibilidade](../articles/active-directory/managed-identities-azure-resources/services-support-managed-identities.md) das identidades gerenciadas do seu recurso e os [problemas conhecidos](../articles/active-directory/managed-identities-azure-resources/known-issues.md) antes de começar.
