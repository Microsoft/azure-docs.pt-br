> [!TIP]
> Armazenamento do Azure suporta autorizar o acesso aos recursos de armazenamento de Blob e fila usando o Azure Active Directory (Azure AD). Autorizando usuários ou aplicativos que usam um token OAuth 2.0 retornado pelo AD do Azure fornece segurança superior e facilidade de uso sobre a autorização de chave compartilhada e compartilhado (SAS) de assinaturas de acesso. Com o Azure AD, não é necessário para armazenar a chave de acesso de conta com seu código e os riscos possíveis vulnerabilidades de segurança.
>
> A Microsoft recomenda usando o Azure AD com seus aplicativos de armazenamento do Azure, quando possível. Para obter mais informações, consulte [autorizar o acesso ao Azure blobs e filas usando o Azure Active Directory](../articles/storage/common/storage-auth-aad.md).
