### <a name="azure-active-directory-authentication"></a>Autenticação do Azure Active Directory

Âncoras espacial do Azure também permite que aplicativos se autentiquem com tokens do usuário do Azure AD (Active Directory). Por exemplo, você pode usar tokens do Azure AD para integrar as âncoras espacial do Azure. Se uma empresa mantém os usuários no AD do Azure, você pode fornecer um token de usuário do Azure AD no SDK do Azure espacial âncoras. Isso permite que você autenticar-se diretamente ao serviço âncoras espacial do Azure para uma conta que faz parte do mesmo locatário do AD do Azure.
