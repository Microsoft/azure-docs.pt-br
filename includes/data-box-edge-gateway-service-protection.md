- Para acessar o serviço do Azure Stack Edge, sua organização precisa ter uma assinatura de Enterprise Agreement (EA) ou provedor de soluções na nuvem (CSP). Para obter mais informações, consulte [inscrever-se para uma assinatura do Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Como esse serviço de gerenciamento é hospedado no Azure, ele é protegido pelos recursos de segurança do Azure. Para obter mais informações sobre os recursos de segurança fornecidos pelo Azure, vá para a [central de confiabilidade do Microsoft Azure](https://azure.microsoft.com/support/trust-center/security/).
- Para operações de gerenciamento de SDK, você pode obter a chave de criptografia para seu recurso nas **Propriedades do dispositivo**. Você poderá exibir a chave de criptografia somente se tiver permissões para o API do Graph de recursos.
