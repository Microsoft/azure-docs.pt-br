- Para acessar o serviço de borda da caixa de dados, sua organização precisa ter uma assinatura do Enterprise Agreement (EA) ou o provedor de solução de nuvem (CSP). Para obter mais informações, consulte [inscrever-se para uma assinatura do Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Como esse serviço de gerenciamento está hospedado no Azure, ele é protegido pelos recursos de segurança do Azure. Para obter mais informações sobre os recursos de segurança fornecidos pelo Azure, vá para o [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Para operações de gerenciamento do SDK, você pode obter a chave de criptografia para o recurso no **propriedades do dispositivo**. Você pode exibir a chave de criptografia somente se você tiver permissões para a API do Graph de recurso.
