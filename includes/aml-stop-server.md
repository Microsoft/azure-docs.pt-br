Caso tenha usado uma instância de computação ou uma VM do Notebook, interrompa a VM quando ela não estiver sendo usada para reduzir o custo.

1. No workspace, selecione **Computação**.

1. Selecione a VM na lista.

1. Selecione **Interromper**.

1. Quando estiver pronto para usar o servidor novamente, selecione **Iniciar**.
