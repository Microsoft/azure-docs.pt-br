## <a name="troubleshooting"></a>Solução de problemas

### <a name="tips"></a>Dicas

Caso o objeto não esteja sendo detectado, você pode tentar as seguintes etapas:

1. Verifique novamente se você está usando o modelo correto para seu objeto.
2. Visualize a área de pesquisa e verifique se ela abrange o objeto de destino.
3. Tente reduzir a `MinSurfaceCoverage`.
4. No Portal de Dispositivos do Windows, clique em **Exibições -> Exibições 3D** e confira se a verificação foi concluída.
