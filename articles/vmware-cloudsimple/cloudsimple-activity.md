# <a name="activity-management-overview"></a>Visão geral do gerenciamento de atividades

O CloudSimple mantém o controle de toda a atividade que pode afetar o funcionamento do seu ambiente de nuvem privada. As atividades incluem alertas, eventos, tarefas e atividade de auditoria. As [páginas de atividade](monitor-activity.md) resumem toda a atividade atual e permitem que você faça uma busca detalhada para obter detalhes adicionais.

## <a name="events"></a>Eventos

Os eventos rastreiam a atividade do usuário e do sistema no portal do CloudSimple. Os eventos mostram a atividade associada a um recurso específico e a gravidade do impacto.  Você pode exibir os eventos no portal do CloudSimple.

## <a name="alerts"></a>Alertas

Alertas são notificações de qualquer atividade significativa em seu ambiente CloudSimple. Os eventos que afetam a cobrança ou o acesso do usuário são mostrados como alertas.  Você pode reconhecer alertas no portal do CloudSimple.

## <a name="tasks"></a>Tarefas

As tarefas rastreiam qualquer operação de usuário que leva mais de 30 segundos para ser concluída. Você pode monitorar o progresso de uma tarefa no portal do CloudSimple.  Para tarefas concluídas, as informações incluem o tempo total para a conclusão.

## <a name="audit"></a>Audit

Os logs de auditoria controlam as operações do usuário. Os logs de auditoria contêm os parâmetros fornecidos para a operação pelo usuário.  Você pode usar os logs de auditoria para monitorar a atividade do usuário para todos os usuários.
