# <a name="cloudsimple-node-quota-limits"></a>Limites de cota de nó CloudSimple

Quatro nós é a quantidade de padrão disponível para provisionamento, quando sua assinatura está habilitada para o serviço CloudSimple.  Você pode provisionar qualquer [tipo de nó](cloudsimple-node.md) do portal do Azure.  Um mínimo de três nós da mesma SKU são necessários para criar uma nuvem privada.  Se você tiver provisionado os nós, você verá um erro ao tentar provisionar nós adicionais.

## <a name="quota-increase"></a>Aumento de cota

Você pode aumentar a cota de nó, enviando uma solicitação de suporte. A equipe de operações de serviço avaliará a solicitação e trabalhar com você para aumentar a cota de nó.  Quando você abre um novo tíquete, selecione as seguintes opções:

* Tipo de problema: **Técnico**
* Assinatura: **Sua ID de assinatura**
* Tipo de serviço: **Solução do VMware CloudSimple**
* Tipo de problema: **Cota de nós dedicados**
* Subtipo de problema: **Aumentar a cota de nós dedicados**
* Assunto: **Aumento de cota**

Nos detalhes de tíquete de suporte, forneça o número necessário de nós e nó de SKU.

Você também pode contatar seu representante de conta da Microsoft em [ azurevmwaresales@microsoft.com ](mailto:azurevmwaresales@microsoft.com) para aumentar a cota de nó em sua assinatura.  Você precisará fornecer a:

* ID da assinatura
* Nó de SKU
* Número de nós adicionais para o qual você está solicitando o aumento de cota
