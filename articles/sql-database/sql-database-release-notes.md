Este artigo lista os recursos de banco de dados SQL que estão atualmente em visualização pública. Para obter atualizações de banco de dados SQL e aprimoramentos, consulte [atualizações de serviço de banco de dados SQL](https://azure.microsoft.com/updates/?product=sql-database). Para atualizações e aprimoramentos para outros serviços do Azure, consulte [atualizações de serviço](https://azure.microsoft.com/updates).
