Este artigo descreve como programar diretamente em relação ao protocolo em seu aplicativo para solicitar tokens do Azure AD.  Quando possível, recomendamos que você use as MSAL (bibliotecas de autenticação da Microsoft) com suporte para [adquirir tokens e chamar APIs Web seguras](..\authentication-flows-app-scenarios.md#scenarios-and-supported-authentication-flows).  Confira também os [aplicativos de exemplo que usam MSAL](..\sample-v2-code.md).
