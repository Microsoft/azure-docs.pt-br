# <a name="sign-in-activity-report-error-codes"></a>Códigos de erro do relatório de atividade de login 

Com as informações fornecidas pelo [relatório de entradas do usuário](concept-sign-ins.md), você encontra respostas para perguntas como:

- Quem entrou na minha inscrição?
- Quais aplicativos foram conectados?
- Quais entradas falharam e por quê?

Quando uma entrada falha, você vê um código de erro correspondente à falha. [Localize a descrição de um código de erro](https://login.microsoftonline.com/error).
