# <a name="access-a-vm-student-view-in-azure-lab-from-teams"></a>Acessar uma VM (exibição de aluno) no laboratório do Azure das equipes

Quando um laboratório é criado dentro das equipes, os usuários podem exibir e acessar todas as VMs provisionadas pelo proprietário da equipe. Depois que o laboratório for publicado e as VMs forem criadas, os usuários serão automaticamente registrados no laboratório e receberão uma VM em seu primeiro logon para Azure Lab Services. Os usuários podem exibir e acessar a VM atribuída a eles selecionando a guia que contém **Azure Lab Services** aplicativo.

Se o laboratório ainda estiver sendo publicado ou se uma sincronização ainda não for disparada depois de ser adicionada à equipe, mensagens semelhantes serão mostradas aos alunos.
