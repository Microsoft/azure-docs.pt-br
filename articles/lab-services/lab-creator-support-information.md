# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>Exibir informações de suporte (criador de laboratório no Azure Lab Services)
Este artigo explica como você (como um criador de laboratório) pode exibir as seguintes informações de suporte:

- URL
- Email
- Telefone
- Instruções adicionais

Você pode usar essas informações para obter ajuda quando encontrar problemas técnicos durante a criação de um laboratório em uma conta de laboratório.

 
## <a name="view-support-information"></a>Exibir informações de suporte
1. Entre no [portal do Azure Lab Services](https://labs.azure.com).
2. Selecione o ponto de interrogação (**?**) no canto superior direito da página. 
3. Confirme que você vê links para **exibir site de suporte**, **suporte por email** e **número de telefone do suporte**.

    ![Exibir informações de suporte](./media/lab-creator-support-information/support-information.png)
