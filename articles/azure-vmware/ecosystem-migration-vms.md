# <a name="migration-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Soluções de migração para VMs (máquinas virtuais) da solução do Azure VMware

Um dos casos de uso mais comuns para usar a solução Azure VMware é data center a evacuação.  Ele permite que você continue a maximizar seus investimentos em VMware, pois a solução Azure VMware estará sempre atualizada. Além disso, você pode aprimorar suas cargas de trabalho com a gama completa de serviços nativos do Azure.  Uma etapa de chave inicial nesse processo é a migração de seu ambiente herdado baseado em VMware para a solução do Azure VMware.

Nossos parceiros de migração têm soluções de migração líderes do setor em ambientes baseados em VMware. Os clientes em todo o mundo usaram essas soluções para suas migrações para a solução Azure e Azure VMware.

Não é necessário usar o VMware HCX como uma ferramenta de migração, o que significa que você também pode migrar cargas de trabalho físicas para a solução VMware do Azure. Além disso, as migrações para seu ambiente de solução do Azure VMware não precisam de uma conexão de ExpressRoute se não estiverem disponíveis no seu ambiente de origem.  As migrações podem ser feitas em vários locais se você decidir hospedar essas cargas de trabalho em várias regiões do Azure.

Para obter mais informações sobre essas soluções, consulte [RiverMeadow](https://www.rivermeadow.com/migrating-to-vmware-on-azure).
