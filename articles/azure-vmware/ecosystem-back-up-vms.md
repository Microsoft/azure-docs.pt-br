# <a name="backup-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Soluções de backup para VMs (máquinas virtuais) da solução do Azure VMware

Um princípio importante da solução Azure VMware é permitir que você continue a usar seus investimentos e suas soluções de VMware favoritas em execução no Azure. O suporte à tecnologia ISV (fornecedor independente de software), validado com a solução Azure VMware, é uma parte importante dessa estratégia. 

Nossos parceiros de backup têm soluções de backup e restauração líderes do setor em ambientes baseados em VMware. Os clientes adotaram amplamente essas soluções para suas implantações locais. Agora, esses parceiros estenderam suas soluções para a solução Azure VMware, usando o Azure para fornecer um repositório de backup e um destino de armazenamento para retenção e arquivamento de longo prazo.

O tráfego de rede de backup entre as VMs de solução do Azure VMware e o repositório de backup no Azure passa por um link de baixa latência e alta largura de banda. O tráfego de replicação entre regiões passa pela rede interna do Azure backplane, o que reduz os custos de largura de banda para os usuários.
