# <a name="azure-kinect-support-options-and-resources"></a>Recursos e opções de suporte do Azure Kinect

Este artigo explicará várias opções de suporte.

## <a name="community-support"></a>Suporte da comunidade

Há várias maneiras de fazer suas perguntas responderem por meio de fóruns públicos:

- [StackOverflow](https://stackoverflow.com/search?q=azurekinect&s=3b855ed0-8564-4961-856f-9614aeab4c0d&s=fd9ea920-622c-4d8e-b908-ec996e1f1403), onde você pode fazer perguntas ou pesquisar por meio de uma biblioteca de perguntas existente.
- [GitHub](https://github.com/Microsoft/Azure-Kinect-Sensor-SDK), onde você pode fazer perguntas, abrir novos bugs ou contribuir para o desenvolvimento do SDK do sensor de Kinect do Azure.
- [Forneça comentários](https://feedback.azure.com/forums/920053-azure-kinect-dk), onde você pode compartilhar suas ideias para o futuro do produto e votar em uma ideia existente.

## <a name="assisted-support"></a>Suporte assistido

Há várias maneiras de obter um suporte assistido para o Azure Kinect.

### <a name="microsoft-qa"></a>P e R da Microsoft

Para obter respostas rápidas e confiáveis para suas perguntas técnicas sobre produtos com os engenheiros da Microsoft, os MVPs (profissionais mais valiosos) do Azure ou nossa comunidade de especialistas, entre em [P e R da Microsoft](/answers/products/azure), o destino preferido do Azure para suporte à comunidade.

- [O Microsoft Q&A para Azure Kinect](/answers/topics/azure-kinect-dk.html), onde você pode fazer perguntas ou pesquisar por meio de uma biblioteca de perguntas existente.

### <a name="development-azure-kinect-on-azure"></a>Desenvolvimento do Azure Kinect no Azure

Os assinantes do Azure podem criar e gerenciar solicitações de suporte no portal do Azure. O suporte de desenvolvimento one-on-one para acompanhamento de corpo, SDK do sensor, fala do dispositivo de voz ou serviços de cognitiva do Azure está disponível para assinantes do Azure com um [plano de suporte do Azure](https://azure.microsoft.com/support/plans/) associado à sua assinatura.

  - Tem um [plano de suporte do Azure](https://azure.microsoft.com/support/plans/) associado à sua assinatura do Azure?  [Entre no portal do Azure](https://ms.portal.azure.com/) para enviar um incidente.
  - Precisa de uma assinatura do Azure? [As opções de assinatura do Azure](https://azure.microsoft.com/pricing/purchase-options/) fornecerão mais informações sobre diferentes opções.
  - Precisa de um plano de suporte? [Selecionar plano de suporte](https://azure.microsoft.com/support/plans/)

### <a name="azure-kinect-on-premises-or-other-cloud-services"></a>Azure Kinect local ou outros serviços de nuvem

Para obter suporte técnico usando o SDK do sensor e o SDK de rastreamento de corpo no local, abra um tíquete para suporte profissional no [portal de suporte da Microsoft](https://support.microsoft.com/supportforbusiness/productselection?sapId=c49ea5bb-2b09-8612-be35-d55159732667).

### <a name="azure-kinect-dk-device"></a>Dispositivo Azure Kinect DK

Antes de entrar em contato com o suporte de hardware, verifique se você configurou e atualizou o Azure Kinect DK. Para testar se o dispositivo está funcionando, use o [Azure Kinect Viewer](azure-kinect-viewer.md). Saiba mais em nossa página de [ajuda do Azure Kinect DK](./index.yml) .
Talvez você também queira conferir nossos [problemas conhecidos e solucionar problemas](troubleshooting.md).

[Obtenha ajuda](https://support.microsoft.com/supportforbusiness/productselection?sapId=f77b1b95-721e-43a0-2db8-b01e81a3f813) com uma funcionalidade de dispositivo ou sensor, atualizações de firmware ou opções de compra.

Para obter mais informações sobre ofertas de suporte, saiba mais no [suporte da Microsoft para empresas](https://support.microsoft.com/help/4341255/support-for-business).

Declarações de conformidade com a UE para produtos de hardware da Microsoft estão [aqui](https://www.microsoft.com/en-us/eu-declarations-compliance?activetab=pivot1:primaryr3)
