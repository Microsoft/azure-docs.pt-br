# <a name="customizing-content-models-in-video-indexer"></a>Personalizando modelos de conteúdo no Video Indexer

Indexador de vídeo permite personalizar alguns dos seus modelos de se adaptar ao seu caso de uso específico. Esses modelos incluem [marcas](customize-brands-model-overview.md), [idioma](customize-language-model-overview.md), e [pessoa](customize-person-model-overview.md). Você pode personalizar facilmente esses modelos usando o site do indexador de vídeo ou a API.

Este artigo fornece links para os artigos conceituais que explicam os benefícios de cada tipo de personalização. Este artigo também fornece links para guias de instruções que mostram como você pode implementar a personalização de cada modelo.
