# <a name="how-to-find-your-azure-tenant-id"></a>Como localizar sua ID de locatário do Azure

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Localizar sua ID de locatário no portal

1. Entre no [portal do Azure](https://portal.azure.com).
1. Selecione o ícone de Azure Active Directory em serviços do Azure. A página de Active Directory será aberta.
1. Sua ID de locatário está localizada em **informações básicas**.
1. Copie a ID do locatário e cole-a em um documento de texto de sua escolha para uso posterior.
