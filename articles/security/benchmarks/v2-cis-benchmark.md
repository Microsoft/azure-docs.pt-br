# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Mapeamento do parâmetro de comparação de segurança do Azure V2 e do CIS Microsoft Azure Foundations

O parâmetro de comparação do CIS Microsoft Azure Foundations é a orientação de segurança fornecida pelo Center for Internet Security para estabelecer uma configuração de linha de base segura para o Azure. O escopo do parâmetro de comparação é estabelecer o nível básico de segurança ao adotar a nuvem do Azure. Esse parâmetro de comparação está em alinhamento com o parâmetro de [comparação de segurança do Azure v2](overview.md).

Fornecemos um mapeamento entre o [CIS Microsoft Azure Foundations benchmark mais recente versão v 1.3.0](https://www.cisecurity.org/benchmark/azure/) e o [benchmark de segurança do Azure v2](overview.md). Isso o ajudará a entender como as recomendações para ambos os parâmetros de comparação estão alinhadas. Você pode baixar o mapeamento completo [aqui](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx).
