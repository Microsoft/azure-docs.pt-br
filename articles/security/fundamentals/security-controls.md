# <a name="index-of-built-in-security-controls-articles-for-azure-services"></a>Índice de artigos "controles de segurança internos" para serviços do Azure

Este índice fornece links para artigos de "controles de segurança internos". Um controle de segurança é uma qualidade ou um recurso de um serviço do Azure que contribui para a capacidade do serviço de prevenir, detectar e responder a vulnerabilidades de segurança.

Os artigos de controle de segurança internos estão disponíveis para os seguintes serviços:


- [Serviço de Aplicativo do Azure](../../app-service/security-baseline.md)
- [Azure Resource Manager](../../azure-resource-manager/management/azure-resource-manager-security-controls.md)
- [Retransmissão do Barramento de Serviço do Azure](../../azure-relay/service-bus-relay-security-controls.md)
- [Azure Spring Cloud](../../spring-cloud/spring-cloud-concept-security-controls.md)
