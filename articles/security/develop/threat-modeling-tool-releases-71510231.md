# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Versão 7.1.51023.1 de atualização da Threat Modeling Tool – 01/11/2018

Como observado originalmente nas [notas sobre a versão de disponibilidade geral](threat-modeling-tool-releases-71509112.md), lançamos uma atualização (7.1.51023.1) da Microsoft Threat Modeling Tool que exigirá que os usuários da versão prévia (clientes da versão prévia com uma versão inferior à 7.1.50911.2) atualizem para a versão de disponibilidade geral com suporte. Essa versão não contém nenhuma funcionalidade ou correção nova.

- Os usuários da versão prévia baixarão automaticamente a atualização quando o cliente for aberto. Se você escolher não instalar a nova atualização, a versão prévia da ferramenta será fechada.
- Os usuários da versão de disponibilidade geral da ferramenta serão solicitados a escolher se desejam ou não atualizar.
- Os usuários da versão de 2016 da ferramenta não serão afetados.

## <a name="feature-changes"></a>Alterações de recurso

- Nenhum

## <a name="system-requirements"></a>Requisitos de sistema

- Sistemas operacionais com suporte
  - Microsoft Windows 10
- Versão necessária do .NET
  - 3.5.2 .NET
- Requisitos adicionais
  - Uma conexão de Internet é necessária para receber as atualizações da ferramenta e também os modelos.
