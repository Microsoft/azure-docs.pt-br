# <a name="azure-arc-enabled-sql-managed-instance-overview"></a>Visão geral da Instância Gerenciada de SQL habilitada para Azure Arc

A Instância Gerenciada de SQL habilitada para Azure Arc é um serviço de dados do SQL Azure que pode ser criada na infraestrutura de sua escolha. 

[!INCLUDE [azure-arc-data-preview](../../../includes/azure-arc-data-preview.md)]

## <a name="description"></a>Descrição

A Instância Gerenciada de SQL habilitada para Azure Arc tem quase 100% de compatibilidade com o mecanismo de banco de dados SQL Server mais recente e permite que os clientes existentes do SQL Server migrem e alterem seus aplicativos para os serviços de dados do Azure Arc com alterações mínimas ao aplicativo e ao banco de dados enquanto mantêm a soberania de dados. Ao mesmo tempo, a Instância Gerenciada de SQL inclui recursos de gerenciamento internos que reduzem drasticamente a sobrecarga de gerenciamento. 

A Instância Gerenciada de SQL habilitada para Azure Arc está em versão prévia.
