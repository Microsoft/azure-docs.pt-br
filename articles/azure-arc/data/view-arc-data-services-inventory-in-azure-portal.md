# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Exibir o inventário de suas instâncias no portal do Azure

Depois de carregar suas [métricas, logs](upload-metrics-and-logs-to-azure-monitor.md)ou [uso](view-billing-data-in-azure.md), você pode exibir sua instância como um recurso do Azure do portal do Azure. Para exibir o recurso no portal do Azure, vá para a URL especial [https://portal.azure.com](https://portal.azure.com) e conclua estas etapas:

1. Vá para **todos os serviços**.
1. Pesquise o tipo de instância do banco de dados.
1. Adicione o tipo aos seus favoritos.
1. No menu à esquerda, selecione o tipo de instância.
1. Exiba suas instâncias na mesma exibição que seus outros recursos de hiperescala do Azure SQL ou PostgreSQL do Azure (Use filtros para uma exibição granular).
