# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Exibir recurso do controlador de dados de arco do Azure no portal do Azure

Depois de fazer suas primeiras [métricas ou carregar logs no Azure](upload-metrics-and-logs-to-azure-monitor.md) ou [carregar o uso](view-billing-data-in-azure.md), você poderá ver que o recurso do controlador de dados Arc do Azure criado no Azure no portal do Azure.

Abra o portal do Azure usando a URL especial:  [https://portal.azure.com](https://portal.azure.com) .

Pesquise por seu controlador de dados por nome na barra de pesquisa e clique nele.
