# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Parceiros de governança, gerenciamento e migração de dados do armazenamento do Azure

Essas soluções de parceiros podem dar suporte à avaliação de armazenamento e relatório, migração independente de plataforma, replicação, camadas de nuvem ou governança de dados.
