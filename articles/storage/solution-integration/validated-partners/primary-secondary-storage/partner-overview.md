# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Parceiros de armazenamento do Azure para armazenamento primário e secundário

Este artigo destaca as empresas parceiras da Microsoft que fornecem uma solução de NAS (armazenamento conectado à rede) ou SAN (rede de área de armazenamento). A solução pode ser local, no Azure ou uma solução híbrida que usa o armazenamento do Azure como uma camada econômica. Essas soluções podem permitir que os clientes usem a mesma solução em qualquer um de seus ambientes.
