# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Parceiros de arquivamento de armazenamento do Azure, backup e recuperação de desastre

Este artigo destaca os parceiros da Microsoft que estão integrados ao armazenamento do Azure para cargas de trabalho de arquivamento e BCDR. Eles aproveitam os benefícios de escala e [custo](https://docs.microsoft.com/azure/storage/blobs/storage-lifecycle-management-concepts?tabs=azure-portal) do armazenamento do Azure para resolver desafios de backup com eficiência, criar um site de recuperação de desastre ou arquivar conteúdo não utilizado para retenção de longo prazo. Com todos os padrões de conformidade que o armazenamento do Azure atende e com o gerenciamento de ciclo de vida e [armazenamento imutável](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage) , essas soluções podem facilmente substituir backups baseados em fita e oferecer um site de recuperação econômico sob demanda.
