# <a name="azure-storage-analytics-and-big-data-partners"></a>Parceiros Análise de Armazenamento do Azure e de Big data

Este artigo destaca as empresas parceiras da Microsoft que estão integradas ao Azure Data Lake Storage. Eles abrangem cargas de trabalho, incluindo data warehouse moderno, análise avançada, análise em tempo real, etc. Esses parceiros estão aproveitando o espaço de nome hierárquico no armazenamento do Azure para otimizar sua solução e executá-la com eficiência no Azure.
