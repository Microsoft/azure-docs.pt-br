# <a name="azure-files-and-azure-netapp-files-comparison"></a>Comparação de arquivos e Azure NetApp Files do Azure

Este artigo fornece uma comparação dos arquivos e Azure NetApp Files do Azure. 

A maioria das cargas de trabalho que exigem o armazenamento de arquivos em nuvem funciona bem em arquivos do Azure ou Azure NetApp Files. Para ajudar a determinar a melhor opção para sua carga de trabalho, examine as informações fornecidas neste artigo. Para obter mais informações, consulte a documentação arquivos e [Azure NetApp files](https://docs.microsoft.com/azure/azure-netapp-files/) do [Azure](https://docs.microsoft.com/azure/storage/files/) .
