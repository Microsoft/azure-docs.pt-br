# <a name="azure-data-factory-whitepapers"></a>White papers de Data Factory do Azure 

White papers permitem que você explore o Azure Data Factory em um nível mais profundo. Este artigo fornece uma lista de white papers disponíveis para o Azure Data Factory.

| **Whitepaper** | **Descrição** |
| --- | --- |
|[Azure Data Factory — integração de dados na nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Este documento descreve como Azure Data Factory permite que você criar um data warehouse moderno, habilite a análise avançada para a unidade de aplicativos inteligentes de SaaS e migre seus pacotes de serviços de integração do SQL Server no Azure.|
|[Migração de dados do Data Warehouse relacional de dados local para o Azure usando o Azure Data Factory](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | Este artigo aborda a complexidade da migração de dezenas de TB de dados de existente de data warehouse relacional no local (por exemplo, Netezza, Oracle, Teradata, SQL server) para o Azure (por exemplo, o armazenamento de BLOBs ou Azure Data Lake Storage) usando o Azure Data Factory. Os desafios e as práticas recomendadas são ilustradas em torno de resiliência, desempenho, escalabilidade, gerenciamento e segurança da jornada de ingestão de grandes dados no Azure pelo Azure Data Factory. |
|[Azure Data Factory: SSIS na nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Este documento passa por que você desejaria migrar suas cargas de trabalho existentes do SSIS para o Azure Data Factory e considerações comuns de endereço e preocupações. Em seguida, vamos orientá-lo por meio de detalhes técnicos da criação de um IR Azure-SSIS e, em seguida, mostrarei como carregar, executar e monitorar seus pacotes por meio do Azure Data Factory usando as ferramentas que você provavelmente estão familiarizadas, como o SQL Server Management Studio (SSMS). |
|[Fábrica de dados do Azure – passando parâmetros](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Este artigo explica como passar parâmetros entre um pipeline e atividades, bem como entre as atividades. |
|[Azure data Factory – DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Este white paper aborda algumas das práticas recomendadas em torno da integração contínua e implantação do Azure Data Factory. |
