# <a name="unity-showcase-application"></a>Aplicativo de demonstração do Unity

O aplicativo de demonstração do Unity é um aplicativo de exemplo baseado em Unity para demonstrar os recursos do Azure Remote Rendering. Ele pode ser executado dentro do editor do Unity e no Hololens 2.

![Aplicativo de demonstração](./media/showcase-app.png "Aplicativo de demonstração")

## <a name="source-repository"></a>Repositório de origem

O código-fonte para este aplicativo, junto com as instruções de compilação/implantação, pode ser encontrado no [repositório de amostras do ARR](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/Showcase).
