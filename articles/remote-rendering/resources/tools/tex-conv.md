# <a name="texconv---texture-conversion-tool"></a>TexConv-ferramenta de conversão de textura

A ferramenta TexConv é uma ferramenta de linha de comando externa para executar conversões de textura em lote.
A ferramenta pode ser encontrada neste repositório git: [repositório GitHub para a ferramenta TexConv](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Para a renderização remota, as texturas não precisam ser convertidas, pois a etapa de conversão pode lidar com formatos de arquivo de imagem comuns e converte texturas no formato de tempo de execução apropriado. No entanto, essa ferramenta pode ser útil para converter texturas localmente com configurações personalizadas.
