#### <a name="c-nuget-package"></a>Pacote NuGet do C#

O SDK de fala do .NET pode ser instalado por meio do **CLI do .NET Core** com o comando a seguir `dotnet add` .

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

O SDK de fala do .NET pode ser instalado por meio do **Gerenciador de pacotes** com o comando a seguir `Install-Package` .

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
