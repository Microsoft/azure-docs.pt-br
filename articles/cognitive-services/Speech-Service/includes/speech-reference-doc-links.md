## <a name="reference-docs"></a>Documentos de Referência

O serviço de fala fornece dois SDKs. O primeiro SDK é o principal [SDK de fala](../speech-sdk.md) e fornece a maioria das funcionalidades necessárias para interagir com o serviço de fala. O segundo SDK é específico para dispositivos, adequadamente nomeados como [SDK de dispositivos de fala](../speech-devices-sdk.md). Ambos os SDKs estão disponíveis em vários idiomas.

### <a name="speech-sdk-reference-docs"></a>Documentos de referência do SDK de fala

Use a lista a seguir para encontrar os documentos de referência do SDK de fala apropriados:

- <a href="https://aka.ms/csspeech/csharpref" target="_blank" rel="noopener">SDK DO C# </a>
- <a href="https://aka.ms/csspeech/cppref" target="_blank" rel="noopener">SDK DO C++ </a>
- <a href="https://aka.ms/csspeech/javaref" target="_blank" rel="noopener">SDK do Java </a>
- <a href="https://aka.ms/csspeech/pythonref" target="_blank" rel="noopener">SDK do Python</a>
- <a href="https://aka.ms/csspeech/javascriptref" target="_blank" rel="noopener">SDK do JavaScript</a>
- <a href="https://aka.ms/csspeech/objectivecref" target="_blank" rel="noopener">SDK do Objective-C </a>

> [!TIP]
> O SDK do serviço de fala é mantido e atualizado ativamente. Para controlar alterações, atualizações e adições de recursos, consulte as [notas de versão do SDK de fala](../releasenotes.md).

### <a name="speech-devices-sdk-reference-docs"></a>Documentos de referência do SDK de dispositivos de fala

O [SDK dos dispositivos de fala](../speech-devices-sdk.md) é um superconjunto do SDK de fala, com funcionalidade estendida para dispositivos específicos. Para baixar o SDK de dispositivos de fala, você deve primeiro [escolher um kit de desenvolvimento](../get-speech-devices-sdk.md#choose-a-development-kit).
