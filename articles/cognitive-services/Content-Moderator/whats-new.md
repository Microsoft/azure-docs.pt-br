# <a name="whats-new-in-content-moderator"></a>Novidades do Content Moderator

Conheça o que há de novo no serviço. Esses itens podem ser notas sobre a versão, vídeos, postagens no blog e outros tipos de informações. Marque esta página para se manter atualizado quanto ao serviço.

## <a name="august-2020"></a>Agosto de 2020

### <a name="new-video-moderation-ui"></a>Nova interface do usuário de moderação de vídeo

* A página de moderação de vídeo da ferramenta de Análise foi atualizada com muitas ferramentas e novos recursos a fim de facilitar a marcação e o processamento de vídeos e trechos de vídeos. Confira o [Guia de moderação de vídeo](./video-moderation-human-review.md) para saber mais.

### <a name="new-dashboard"></a>Novo painel

* O painel da ferramenta de Análise foi atualizado para mostrar mais informações sobre as análises em aberto, pendentes e concluídas. Confira o [guia da ferramenta de Análise](./Review-Tool-User-Guide/human-in-the-loop.md) para fazer um tour pelos novos recursos.
