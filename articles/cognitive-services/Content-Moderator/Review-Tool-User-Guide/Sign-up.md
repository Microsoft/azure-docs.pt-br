# <a name="sign-up-for-content-moderator"></a>Inscrever-se no Content Moderator

Inscreva-se em Content Moderator [ferramenta de revisão](https://contentmoderator.cognitive.microsoft.com) site. Você pode inscrever-se com sua conta da Microsoft ou criar uma conta no site.

![Inscrição](images/0-signup-1.PNG)

## <a name="name-your-team"></a>Nomeie sua equipe

Forneça um nome para a equipe de revisão. Opcionalmente, você pode convidar seus colegas para se juntarem à equipe.

![Criar equipe](images/1-Team-Name.PNG)
