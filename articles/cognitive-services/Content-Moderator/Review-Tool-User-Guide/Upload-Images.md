# <a name="upload-images"></a>Carregar imagens

Para explorar a interface do usuário, carregue até cinco imagens por vez, cada imagem não deve exceder 2 MB de tamanho de arquivo. Certifique-se de verificar os tamanhos de arquivo antes de carregar.

![Carregar imagens](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Enviar para moderação
Se você carregar mais de cinco imagens por vez, use o ícone de Lixeira para remover as imagens indesejadas. Em seguida, envie as imagens de moderação automatizada.

![Enviar imagens](images/1-try-image-2.png)
