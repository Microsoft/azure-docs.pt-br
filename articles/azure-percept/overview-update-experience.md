# <a name="azure-percept-dk-update-experience-overview"></a>Visão geral da experiência de atualização do Azure Percept DK

Com o Azure Percept DK, você pode atualizar seu sistema operacional do kit de desenvolvimento e o firmware over-the-Air (OTA) ou via USB. A atualização OTA é uma maneira fácil de manter os dispositivos atualizados por meio da [atualização do dispositivo para o serviço Hub IOT](https://docs.microsoft.com/azure/iot-hub-device-update/) . Atualizações de USB estão disponíveis para usuários que não podem usar atualizações OTA ou quando uma redefinição de fábrica do dispositivo é necessária. Confira os seguintes guias de instruções para começar a usar as atualizações de dispositivo do Azure Percept DK:

- [Configurar o Hub IoT do Azure para implantar atualizações OTA (over-the-Air) no Azure Percept DK](./how-to-set-up-over-the-air-updates.md)
- [Atualizar seu Percept do Azure (OTA) por satélite](./how-to-update-over-the-air.md)
- [Atualizar seu Percept do Azure DK em USB](./how-to-update-via-usb.md)
