# <a name="the-azure-percept-dk-update-experience"></a>A experiência de atualização do Azure Percept DK

Com o Azure Percept DK, você tem duas opções para atualizar o sistema operacional e o firmware do seu kit de desenvolvimento: OTA (over-the-Air) ou via USB. OTA é a melhor maneira de manter os dispositivos atualizados em escala, enquanto a atualização de USB é uma boa opção para quando o OTA não é possível ou quando você deseja redefinir o dispositivo de fábrica.

## <a name="update-experience-guides"></a>Atualizar guias de experiência

Para garantir que você possa aproveitar qualquer método de atualização ideal para você, reunimos uma coleção de guias de instruções para ajudá-lo.

- [Como configurar o Hub IoT do Azure para implantar as atualizações do ar no Azure Percept DK](./how-to-set-up-over-the-air-updates.md)
- [Como atualizar seu Percept do Azure DK pelo ar](./how-to-update-over-the-air.md)
