# <a name="advanced-development-with-azure-percept"></a>Desenvolvimento avançado com o Azure Percept
Com o Azure Percept, os desenvolvedores de software e os cientistas de dados são capazes de usar fluxos de trabalho de código avançado para o gerenciamento do ciclo de vida de ia. Por meio de uma biblioteca de software livre crescente, eles podem usar exemplos para começar com sua jornada de desenvolvimento de ia e criar soluções prontas de produção.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Introdução aos tutoriais de desenvolvimento avançados

Este [Notebook Jupyter](https://github.com/microsoft/Project-Santa-Cruz-Private-Preview/blob/main/Sample-Scripts-and-Notebooks/Official/Machine%20Learning%20Notebooks/Transferlearningusing_SSDLiteV2%20Model.ipynb) executa aprendizado de transferência, usando um modelo de TensorFlow (MobileNetSSDV2Lite) pré-treinado no AzureML em Python com um conjunto de informações personalizado para detectar os outros. Há duas opções de tutorial para orientá-lo no trabalho com o bloco de anotações para treinar e implantar o nosso detector no projeto de Santa Cruz devkit:

- [Desenvolvimento em nuvem](./advanced-development-cloud.md): neste tutorial, você executará o bloco de anotações no [portal de Azure Machine Learning](https://ml.azure.com) com uma instância de computação remota.

- [Desenvolvimento local](./advanced-development-local.md): neste tutorial, você executará o bloco de anotações localmente no vs Code com uma instância de computação remota.
