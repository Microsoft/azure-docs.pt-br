# <a name="advanced-development-with-azure-percept"></a>Desenvolvimento avançado com o Azure Percept

Com o Azure Percept, os desenvolvedores de software e os cientistas de dados podem usar fluxos de trabalho de código avançado para gerenciamento do ciclo de vida de ia. Por meio de uma biblioteca de software livre crescente, eles podem usar exemplos para começar com sua jornada de desenvolvimento de ia e criar soluções prontas para a produção.

## <a name="get-started-with-advanced-development"></a>Introdução ao desenvolvimento avançado

Consulte o [GitHub de desenvolvimento avançado do Azure PERCEPT DK](https://github.com/microsoft/azure-percept-advanced-development) para obter orientações atualizadas, tutoriais e exemplos para coisas como:

- Implantando um modelo de ia personalizado em seu Azure Percept DK
- Atualizando um modelo com suporte com o aprendizado de transferência
- E mais

## <a name="next-steps"></a>Próximas etapas

Saiba mais sobre os [modelos disponíveis do Azure PERCEPT ia](./overview-ai-models.md). Se nenhum desses modelos atender às suas necessidades, use a jornada de código avançado para trazer seu próprio pipeline de visão de seu modelo ou computador para o Percept DK. Se você tiver uma contribuição que imagina para outras pessoas, sinta-se livre abrir uma solicitação pull também.
