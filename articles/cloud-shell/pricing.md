# <a name="pricing"></a>Preços

## <a name="compute-cost"></a>Custo de computação
O Azure Cloud Shell é executado em um computador fornecido gratuitamente pelo Azure, mas exige um compartilhamento de arquivos do Azure para ser usado. 

## <a name="storage-cost"></a>Custo de armazenamento
O Cloud Shell requer que um compartilhamento de arquivo do Azure novo ou existente seja montado para manter os arquivos entre sessões. O Armazenamento incorre em custos regulares.

Clique em [aqui para obter detalhes sobre os custos de arquivos do Azure](https://azure.microsoft.com/pricing/details/storage/files/).
