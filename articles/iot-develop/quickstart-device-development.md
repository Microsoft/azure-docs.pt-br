# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Introdução ao desenvolvimento de dispositivos inseridos da Internet das Coisas do Azure

**Aplica-se a**: [desenvolvimento de dispositivos inseridos](about-iot-develop.md#embedded-device-development)

Este guia de introdução contém um conjunto de inícios rápidos que mostra como começar a trabalhar com dispositivos inseridos e a Internet das Coisas do Azure. 

Em cada início rápido, você conclui as seguintes tarefas básicas:
* Instalar um conjunto de ferramentas de desenvolvimento inserido para programar um dispositivo específico em C
* Compilar uma imagem que inclua os componentes e exemplos do Azure RTOS e instalar a imagem de um dispositivo
* Conectar com segurança um dispositivo à Internet das Coisas do Azure
* Exibir telemetria do dispositivo, exibir propriedades e invocar métodos de nuvem para dispositivo
