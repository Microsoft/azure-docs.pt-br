# <a name="azure-data-science-virtual-machine-release-notes"></a>Notas sobre a versão da Máquina Virtual de Ciência de Dados do Azure

Neste artigo, saiba mais sobre as versões da Máquina Virtual de Ciência de Dados do Azure. Para ver a lista completa das ferramentas incluídas, juntamente com os números de versão, confira [esta página](./tools-included.md).

Veja [a lista de problemas conhecidos](reference-known-issues.md) para aprender sobre erros e soluções conhecidas.

## <a name="2020-02-24"></a>2020-02-24

Imagens da Máquina Virtual de Ciência de Dados para o [Ubuntu 18.04](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.ubuntu-1804?tab=Overview) e o [Windows 2019](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.dsvm-win-2019?tab=Overview) agora estão disponíveis.

## <a name="2019-10-08"></a>08-10-2019

### <a name="updates-to-software-on-the-windows-dsvm"></a>Atualizações do software no DSVM do Windows

- Gerenciador de Armazenamento do Azure 1.10.1
- Power BI Desktop 2.73.55xx
- Firefox 69.0.2
- PyCharm 19.2.3
- RStudio 1.2.50xx

### <a name="default-browser-for-windows-updated"></a>Navegador padrão do Windows atualizado

Anteriormente, o navegador padrão era o Internet Explorer. Agora, os usuários devem escolher um navegador padrão quando fazem logon pela primeira vez.
