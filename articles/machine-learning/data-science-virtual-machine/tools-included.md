# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Quais ferramentas estão incluídas no Máquina Virtual de Ciência de Dados do Azure?

A Máquina Virtual de Ciência de Dados é uma maneira fácil de explorar dados e fazer aprendizado de máquina na nuvem. As Máquinas Virtuais de Ciência de Dados são pré-configuradas com o sistema operacional completo, patches de segurança, drivers e software popular de desenvolvimento e ciência de dados. Você pode escolher o ambiente de hardware, desde computadores de menor custo centrados na CPU até computadores muito avançados com várias GPUs, armazenamento de NVMe e grandes quantidades de memória. Para computadores com GPUs, todos os drivers são instalados, todas estruturas de aprendizado de máquina são compatíveis com a versão para fornecer compatibilidade com a GPU e a aceleração é habilitada em todo software de aplicativo compatível com GPUs.

O Máquina Virtual de Ciência de Dados vem com as ferramentas de ciência de dados mais úteis pré-instaladas.

## <a name="how-to-choosecondaenvironment"></a>Como escolher o ambiente Conda  

> [!NOTE]
> Há alterações significativas ao alternar para a nova versão da Máquina Virtual de Ciência de Dados da versão antiga. 

**Como alternar do Windows Server 2016 para o Windows Server 2019** </br> 

* O ambiente Conda é definido como python37_default. Não há suporte para python2. Se você estivesse usando TensorFlow ou pytorch em seu ambiente de Conda do Python, use py37_tensorflow ou py37_pytorch, respectivamente. Se você estiver usando somente o AzureML, use azureml_py36_automl </br>

* Se você estivesse usando TensorFlow ou pytorch em seu ambiente Conda do AzureML, use azureml_py36_tensorflow ou azureml_py36_pytorch, respectivamente. </br>

**Como alternar do Ubuntu 16.04 para o Ubuntu 18.04** </br>

* Se você estivesse usando TensorFlow ou pytorch em seu ambiente azureml_py36 Conda, use azureml_py36_tensorflow ou azureml_py36_pytorch, respectivamente. </br>

* Se você estivesse usando TensorFlow ou pytorch em seu ambiente py37 Conda, use py37_tensorflow ou py37_pytorch, respectivamente. </br>

* Caso contrário, use python37_default ou, se você estiver usando somente azureml_py36, use azureml_py36_automl </br>
