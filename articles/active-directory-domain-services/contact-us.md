# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services: entre em contato conosco

## <a name="feedback"></a>Comentários
Se você tiver comentários para nossa perguntas não técnicas ou serviço, você pode compartilhar sua solicitação com o grupo de produto do serviço de domínio do AD do Azure enviando-nos em: [Comentários sobre o Azure AD Domain Services](mailto:aaddsfb@microsoft.com). Emails para esse endereço são revisadas. Os membros do grupo de produtos **talvez** responder a solicitar mais informações. Solicitações de suporte técnico para esse endereço de email não serão respondidas.

## <a name="technical-assistance"></a>Assistência técnica
Se você tiver problemas com seu domínio gerenciado, verifique se as etapas descritas no [Guia de solução de problemas](troubleshoot.md) resolvem o problema. Se você ainda estiver tendo problemas, entre no portal do Azure. Abra o **Azure AD Domain Services** página inicial e selecione **nova solicitação de suporte** do painel de ação em **suporte + solução de problemas**.

![nova solicitação de suporte](./media/contact-us/supportRequest.png) 

Preencha a cada seção da solicitação de suporte mais precisa possível. Ao escolher um serviço, verifique se você selecionar **Azure Active Directory Domain Services (VM - controladores de domínio)** para rotear a solicitação para o profissional de suporte adequado.

> [!IMPORTANT]
> Selecione **objetos, domínios e diretórios do Azure Active Directory do** nas listas de serviços para obter ajuda com a validação de domínio com o Azure Active Directory.
>
>

Essas solicitações de revisão de membros do grupo de produtos.
