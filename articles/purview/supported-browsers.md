# <a name="supported-browsers"></a>Navegadores com suporte 

O Azure alcance dá suporte aos seguintes navegadores. É recomendável que você use o navegador mais atualizado compatível com o sistema operacional. 

* Microsoft Edge (última versão)
* Safari (última versão, apenas Mac)
* Chrome (última versão)
* Firefox (última versão)

## <a name="chrome-incognito-mode"></a>Modo Incognito do Chrome

 O Chrome Incognito bloqueando cookies de terceiros deve estar desabilitado para que o alcance Studio funcione.

:::image type="content" source="./media/supported-browsers/incognito-chrome.png" alt-text="Captura de tela mostrando o Chrome.":::

## <a name="chromium-edge-inprivate-mode"></a>Modo InPrivate do Chromium Edge

O Chromium Edge unprivate usando a prevenção de rastreamento estrito deve ser desabilitado para que o alcance Studio funcione.

:::image type="content" source="./media/supported-browsers/incognito-edge.png" alt-text="Captura de tela mostrando borda.":::
