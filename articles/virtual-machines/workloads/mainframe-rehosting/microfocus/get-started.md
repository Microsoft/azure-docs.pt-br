# <a name="micro-focus-devtest-environments-on-azure"></a>Ambientes de desenvolvimento/teste micro Focus no Azure

A introdução ao Azure com um ambiente de desenvolvimento e teste é uma ótima maneira de impulsionar seus esforços de modernização. Aplicativos de mainframe que dependem de sistemas herdados, como IBM CICS (sistema de controle de informações do cliente) e IMS (sistema de gerenciamento de informações), podem ser rehospedados no Azure usando soluções de parceiros.

O micro Focus fez parceria com a Microsoft para oferecer ferramentas de rehospedagem e migração de mainframe atraentes. Você pode executar aplicativos de mainframe em cenários de desenvolvimento, teste e produção no Azure e na plataforma Microsoft Windows.

O micro Focus oferece rehospedagem de desenvolvimento e soluções de rehospedagem de teste que estendem os aplicativos de negócios COBOL e PL/I existentes para o Azure.
