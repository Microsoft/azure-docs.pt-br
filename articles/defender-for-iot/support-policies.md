# <a name="versioning-and-support-for-azure-defender-for-iot"></a>Controle de versão e suporte do Azure defender para IoT 

Este artigo descreve o suporte, as políticas de alteração significativa para o defender para IoT e as versões do Azure defender para IoT que estão disponíveis no momento. 

## <a name="servicing-information-and-timelines"></a>Informações e cronogramas de manutenção 

A Microsoft planeja lançar atualizações para o Azure defender para IoT não menos de uma vez por trimestre. Cada versão de GA (disponibilidade geral) do Azure defender para o sensor IoT e o console de gerenciamento do Azure defender para IoT no local tem suporte por até nove meses após seu lançamento. Correções e novas funcionalidades serão aplicadas à versão atual do GA que estão atualmente em suporte e não serão aplicadas a versões mais antigas do GA.

## <a name="versions-and-support-dates"></a>Versões e datas de suporte

| Versão | Data de lançamento | Data de término do suporte |
|--|--|--|
| 10.0 | 01/2021 | 10/2021 |
