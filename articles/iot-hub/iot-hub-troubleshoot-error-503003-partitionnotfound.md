# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

Este artigo descreve as causas e soluções para erros de **503003 PartitionNotFound** .

## <a name="symptoms"></a>Sintomas

As solicitações ao Hub IoT falham com o erro **503003 PartitionNotFound**.

## <a name="cause"></a>Causa

Esse erro é interno ao Hub IoT e provavelmente é transitório. Consulte [causa de erro do servidor interno do Hub IOT](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Solução

Consulte [soluções para erros de servidor interno do Hub IOT](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
