# <a name="samples"></a>Exemplos

Serviços de Comunicação do Azure têm muitos exemplos disponíveis, que podem ser usados para testar serviços e recursos do ACS antes de criar seu aplicativo ou caso de uso.

## <a name="application-samples"></a>Exemplos de aplicativo

| Nome do exemplo                                                                                          | Description                                                         | Linguagens/plataformas disponíveis      |
| :--------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------ | :--------------------------------- |
| [Exemplo de Destaque de Chamada de Grupo](./calling-hero-sample.md)                                                | Fornece um exemplo de criação de um aplicativo de chamada de grupo.          | Web, iOS                           |
| [Exemplo de Chamada da Web](./web-calling-sample.md)                                                        | Uma explicação passo a passo dos recursos de Chamada do ACS na Web. | Web                                |
| [Exemplo de Destaque de Chat](./chat-hero-sample.md)                                                            | Fornece um exemplo de criação de um aplicativo de chat.                   | Web, C# e .NET                      |
| [Aplicativo Médico da Contoso](https://github.com/Azure-Samples/communication-services-contoso-med-app)       | Aplicativo de exemplo que demonstra um fluxo entre paciente e médico.                     | Web e Node.js                      |
| [Aplicativo de Varejo da Contoso](https://github.com/Azure-Samples/communication-services-contoso-retail-app)     | Aplicativo de exemplo que demonstra um fluxo de suporte de varejo.                     | ASP.NET, .NET Core, JavaScript/Web |
| [Exemplo de Chamada da WPF](https://github.com/Azure-Samples/communication-services-web-calling-wpf-sample) | Aplicativo de exemplo para Windows que demonstra a funcionalidade de chamada          | WPF / Node.js                      |
