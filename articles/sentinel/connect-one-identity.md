# <a name="connect-one-identity-safeguard"></a>Conectar o One Identity Safeguard
O conector de dados One Identity Safeguard aprimora o conector CEF (Formato Comum de Evento) padrão com dashboards específicos de Proteção de Sessões Privilegiadas. Use esse conector para começar facilmente a usar os eventos gerados pelo seu dispositivo para visualização, alertas, investigações e muito mais.


## <a name="prerequisites"></a>Pré-requisitos

- Permissões de leitura e gravação no workspace do Azure Sentinel.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Conectar o Azure Sentinel ao One Identity Safeguard

1. No portal do Azure, navegue para **Conectores de Dados do** > **Azure Sentinel** e selecione o conector do **One Identity Safeguard**.

2. Clique em **Abrir página do conector**.

3. Siga as instruções na página do **One Identity Safeguard**.
