# <a name="connect-extrahop-revealx"></a>Conectar ExtraHop Reveal(x)

O conector de dados ExtraHop Reveal (x) permite que você conecte facilmente seu sistema revelar (x) com o Azure Sentinel para exibir painéis, criar alertas personalizados e melhorar a investigação. Essa integração oferece a capacidade de obter informações sobre a rede da sua organização e melhorar seus recursos de operação de segurança.


## <a name="prerequisites"></a>Pré-requisitos

- Permissões de leitura e gravação para o espaço de trabalho do Azure Sentinel.

- ExtraHop Discover ou dispositivo de comando com a versão de firmware 7,8 ou posterior com uma conta de usuário que tenha privilégios ilimitados (de administrador).

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Conectar o Azure Sentinel ao ExtraHop Reveal (x)

1. Na portal do Azure, navegue até conectores de dados **do Azure Sentinel**  >  **Data connectors** e selecione o conector **ExtraHop Reveal (x)** .

2. Selecione a **página abrir conector**.

3. Siga as instruções na página **ExtraHop Reveal (x)** .
