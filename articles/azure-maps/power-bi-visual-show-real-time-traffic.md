# <a name="show-real-time-traffic"></a>Mostrar tráfego em tempo real

O recurso de camada de tráfego sobrepõe dados de tráfego em tempo real sobre o mapa. Para habilitar esse recurso, mova o controle deslizante **camada de tráfego** no painel **Formatar** para a posição **ativado** . Isso irá sobrepor os dados do fluxo de tráfego como estradas codificadas por cor.

> [!div class="mx-imgBorder"]
> ![Um mapa que exibe dados de tráfego em tempo real](media/power-bi-visual/traffic-layer.png)

As configurações a seguir estão disponíveis na seção **camada de tráfego** .

| Configuração         | Descrição    |
|-----------------|----------------|
| Mostrar incidentes  | Especifica se os incidentes de tráfego, como fechamentos de estrada e construção, devem ser exibidos no mapa. |
| Controle de tráfego | Adiciona um botão ao mapa que permite aos leitores de relatório ativar ou desativar a camada de tráfego.  |
