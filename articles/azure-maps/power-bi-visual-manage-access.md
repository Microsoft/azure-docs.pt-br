# <a name="manage-the-azure-maps-visual-within-your-organization"></a>Gerenciar o Visual do Azure Maps em sua organização

Power BI fornece a capacidade para designers e administradores de locatário gerenciarem o uso do Visual Maps do Azure. Abaixo, você encontrará as etapas que cada função pode executar.

### <a name="designer-options"></a>Opções do designer

No Power BI Desktop, os designers podem desabilitar o Visual mapas do Azure na guia segurança. Selecione  &gt; **Opções de arquivo e configurações** e, em seguida, selecione **Opções** &gt; **recursos de visualização**. Quando desabilitado, o Azure Maps não será carregado por padrão.  
  
![Painel de opções de Power BI mostrando opções de visualização de recursos para o Visual do Azure Maps](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>Opções do administrador de locatários

No PowerBI.com, os administradores de locatários podem desativar o Visual do Azure Maps para todos os usuários. Selecione **configurações** &gt;  **portal** do administrador &gt; **configurações de locatário**. Quando desabilitado, Power BI não exibirá mais o Visual Maps do Azure no painel visualizações.

> [!div class="mx-imgBorder"]
> ![Power BI portal de administração mostrando as configurações de locatário para o Visual do Azure Maps](media/power-bi-visual/tenant-admin-settings.png)
