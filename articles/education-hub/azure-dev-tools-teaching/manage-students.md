# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Como inscrever os alunos no Azure Dev Tools for Teaching
Após a compra do Azure Dev Tools for Teaching, todos os alunos localizados no domínio da sua instituição terão acesso automaticamente a essa oferta. Durante o processo de compra, você precisará inserir o endereço de email do Administrador da Assinatura. Esse domínio confirmado pela universidade desbloqueará os alunos com o mesmo domínio de email e dará a eles a capacidade de fazer logon e resgatar os respectivos benefícios de download de software. Se os alunos ainda não tiverem uma MSA (conta Microsoft) nem fizerem parte de um Active Directory de Universidade, eles precisarão criar uma que corresponda ao respectivo email da universidade. Se os alunos estiverem tendo problemas para acessar o Hub de Educação, peça que examinem nosso Guia de Solução de Problemas abaixo ou entrem em contato conosco pelo site [https://aka.ms/studentsupport](https://aka.ms/studentsupport).

## <a name="managing-access-for-students"></a>Como gerenciar o acesso para alunos
Você pode gerenciar o acesso para alunos com base no domínio que eles usam para fazer logon no Azure Dev Tools for Teaching. Acesse o [Portal de Gerenciamento](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) do Azure Dev Tools for Teaching. Nele, você verá todas as suas assinaturas. Escolha a assinatura da qual deseja gerenciar o acesso.
