# <a name="what-is-the-azure-education-hub"></a>O que é o Hub de Educação do Azure?

O [Hub de Educação](https://portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/quickstart) do Microsoft Azure é uma ferramenta que ajuda os usuários acadêmicos a provisionar e gerenciar crédito de nuvem em muitas assinaturas diferentes do Azure. Isso é útil quando os membros do corpo docente precisam gerenciar muitos projetos de alunos baseados em nuvem. O Hub de Educação também é útil para fins de pesquisa quando suas necessidades de crédito Azure ainda não são conhecidas.

Os usuários podem ajustar com facilidade o valor de crédito alocado em cada assinatura ou em massa. As assinaturas também podem ser suspensas, por exemplo, quando um projeto semestral ou trimestral é concluído, para maximizar os créditos disponíveis.

## <a name="prerequisites"></a>Pré-requisitos

Para acessar o Hub de Educação, você já deverá ter recebido uma notificação por email informando que você foi aprovado para um auxílio acadêmico. O email contém seu valor de crédito aprovado e um link para o [Hub de Educação](https://aka.ms/startedu).

Para usar seu auxílio acadêmico, você precisará criar uma sala de aula no Hub de Educação e usar assinaturas dentro da sala de aula que acessarão seu crédito Azure.
