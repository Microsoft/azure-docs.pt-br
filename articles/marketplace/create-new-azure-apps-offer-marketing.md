# <a name="how-to-sell-your-azure-application-offer"></a>Como vender sua oferta de Aplicativo Azure

Este artigo descreve as opções adicionais que você pode escolher se estiver vendendo sua oferta de Aplicativo Azure pela Microsoft. Você pode comercializar sua oferta com a Microsoft, revendê-la por meio do [programa CSP (provedores de soluções de nuvem)](cloud-solution-providers.md)ou ambos.

## <a name="co-sell-with-microsoft"></a>Vendas conjuntas com a Microsoft

Fornecer informações sobre a **co-vender com** a guia Microsoft é totalmente opcional. Mas é necessário alcançar o status pronto para venda e _pronto_ para venda de _IP_ . As equipes de vendas da Microsoft usam essas informações para saber mais sobre sua solução ao avaliar seu ajuste para as necessidades do cliente. As informações fornecidas nesta guia não estão disponíveis diretamente para os clientes.

Para obter detalhes e instruções para configurar a guia **co-venda com a Microsoft** , consulte a [opção de venda conjunta no Marketplace comercial](commercial-marketplace-co-sell.md).

## <a name="resell-through-csps"></a>Revenda por meio de CSPs

Se você optar por disponibilizar sua oferta no programa CSP (provedor de soluções de nuvem), os CSPs poderão vender seu produto como parte de uma solução agrupada para seus clientes. Para obter mais informações sobre como revender sua oferta por meio do programa CSP e as instruções passo a passo para configurar a guia **revender por meio de CSPs** , consulte [programa do provedor de soluções de nuvem](cloud-solution-providers.md).
