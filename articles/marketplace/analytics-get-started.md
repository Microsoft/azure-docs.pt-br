# <a name="get-started-with-programmatic-access-to-analytics-data"></a>Introdução ao acesso programático a dados de análise

Este guia ajuda você a se familiarizar com o acesso programático a dados de análise do Marketplace comercial. Você aprenderá como acessar programaticamente relatórios analíticos para monitorar as vendas, avaliar o desempenho e otimizar as ofertas no mercado comercial. As ferramentas de análise aprimoradas permitem que você atue nos resultados de desempenho e mantenha relações melhores com seus clientes e revendedores.

## <a name="about-this-guide"></a>Sobre este guia

Você pode usar este guia para acessar dados de análise do Marketplace comercial programaticamente. Usando os métodos e as APIs documentados neste guia, você pode agendar relatórios personalizados e ingerir conjuntos de dados de chave em seus sistemas de análise internos. Você pode efetivamente monitorar as vendas, avaliar o desempenho e otimizar suas ofertas no mercado comercial.

A API para acessar relatórios do Marketplace comercial permite agendar relatórios personalizados de seus dados de análise de forma assíncrona. O recurso permite definir consultas/modelos de relatórios com base em suas necessidades, definir um agendamento e obter relatórios confiáveis e oportunos em intervalos agendados.

O principal valor de acesso programático a dados de análise do Marketplace comercial é o relatório e a integração personalizados com plataformas e sistemas de BI internos.

> [!NOTE]
> Os parceiros precisarão de recursos de engenharia dedicados para fazer uma integração na interface de API.
