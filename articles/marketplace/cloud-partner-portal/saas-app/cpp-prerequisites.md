# <a name="saas-application-publishing-prerequisites"></a>Pré-requisitos de publicação do Aplicativos SaaS

Esse artigo descreve os pré-requisitos para a publicação de uma oferta do Aplicativo SaaS no Azure Marketplace.  Se você ainda não fez isso, examine os [guia de publicação da oferta de aplicativos de SaaS](../../marketplace-saas-applications-technical-publishing-guide.md). 


## <a name="publishing-prerequisites"></a>Pré-requisitos de publicação

Para publicar uma nova oferta do Aplicativo SaaS, você deve cumprir os seguintes pré-requisitos:
- Acesso a [Portal do Cloud Partner](https://cloudpartner.azure.com/).  Portal do Cloud Partner fornece acesso baseado em função, permitindo que várias pessoas colaborar em direção a publicação de uma oferta. 
- Antes de publicar uma oferta em nome de uma conta do publicador, uma das pessoas com função de Proprietário precisa concordar com os [Termos de Uso](https://azure.microsoft.com/support/legal/website-terms-of-use/), a [Política de Privacidade da Microsoft](https://www.microsoft.com/privacystatement/default.aspx) e o [Contrato do Programa Microsoft Azure Certified](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/).
- Todo o conteúdo do inglês deve ser acompanhado por uma versão em inglês. Esse conteúdo inclui texto, documentos, capturas de tela, Termos de Uso e Política de Privacidade do Storefront. É aceitável fornecer uma URL de link útil para um aplicativo diferente do idioma inglês. 
- Tenha seus metadados prontos para uso. A lista a seguir (não exaustiva) mostra um exemplo desses metadados:
  - Um título
  - Uma descrição (no formato HTML)
  - Uma imagem de logotipo (no formato PNG) e nesses tamanhos de imagem fixos: 40 x 40 pixels, 90 x 90 pixels, 115 x 115 pixels e 255 x 115 pixels.
  - Um *termos de uso* e *política de privacidade* instruções
  - Documentação do aplicativo
  - Contatos de suporte


## <a name="next-steps"></a>Próximas etapas

Depois que esses pré-requisitos foram atendidos, você pode [criar sua oferta de SaaS](./cpp-create-offer.md).
